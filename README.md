# WiFi密码查看
查看本机WiFi密码的小工具（需要ROOT权限），仅支持Android5.0(SdkVersion>21)以上设备，纯练手用。
0权限，干净无广告，安装包只有24KB大，安装完仅占用约100KB的空间。
需要ROOT权限是因为读取WiFi密码所在的文件所需。
